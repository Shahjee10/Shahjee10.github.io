Movies App

A cross-platform mobile app that lets users explore movies via the TMDB API with features like sign-up/login, OTP verification, password reset, multilingual support (English & Urdu), genre-based filtering, and a personal watchlist. The app includes infinite scroll, light/dark mode toggle, and detailed movie pages with posters and trailers. Admins can securely log in to view user details, roles, feedback, and overall watchlist stats. The app showcases real-time data handling, localization, and responsive mobile UI built with React Native.

Role: Full Stack Developer

Tech Stack: React Native · Node.js · Express · MongoDB · JWT · Axios · TMDb API · Tailwind CSS · Multer · Nodemailer
